# Завдання

## Рівень 1
5. Написати сайт курсу валют, в якого буде можливість додавати нові курси співвідношення валюти до гривні. Для кожної валюти має бути два значення курсу купівлі та курсу продажу. В результаті отримуємо таблицю із валютами, яка буде мати 3 колонки: назву, курс купівлі та курс продажу.

## Рівень 2
5. Додайте можливість збереження результатів поточного дня в окремій вкладці.

## Рівень 3
5. Додайте можливість додавання нових валют за допомогою вбудованою адмін-панелі.

## Рівень 4
5. Забезпечте доступ по API задля відображення актуального на сьогодні курсу валют.
